About

Ideaborrrrred is a lightweight, mobile-friendly application designed to help users browse Pinterest content based on interests, ideas, and aesthetics. It’s currently under development and uses the official Pinterest API to provide a smooth and fast experience.

Data Usage

This app does not store any personal information. We only use Pinterest's public APIs to retrieve and display content based on user queries.
